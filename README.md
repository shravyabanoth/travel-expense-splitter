# Travel Expense Splitter

A modern React-based travel expense splitter that helps groups fairly divide expenses with smart settlement calculation.

## Features

- Multiple expenses per person
- Category-wise expense tracking
- Dark / Light mode
- Smart payment settlement system
- Clean and responsive UI
- Real-time expense calculation

## Categories Supported

- Food
- Hotel
- Travel
- Fuel
- Shopping
- Other

## Tech Stack

- React.js
- JavaScript
- CSS
- HTML

## How It Works

1. Enter the number of people
2. Add each person's expenses
3. Select expense categories
4. Calculate fair expense split
5. View settlement details
